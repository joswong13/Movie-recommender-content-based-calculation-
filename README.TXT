The tmdb_5000_movies.csv contains the movies information, most movies span only one line and some movies span multiple lines.
Readmoviesdb.java contains functions to parse the movie database that is stored in a csv file and retain only the necessary information
for the recommender system to function (such as movie name, genre, etc). The moviegui.java file contains the gui to display the list of 
movies to the user, asks for a rating and can recommend movies to users after 5 movies using the content based approach calculations.
